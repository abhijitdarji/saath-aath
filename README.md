# saath-aath
Popular Indian Card Game 78.

Inspired From https://github.com/adamweeks/angular-blackjack

###Technologies used:

* Angular JS
* Angular Material
* Angular Material Font Icons

###Running the application:
````shell
git clone http://github.com/abhijitdarji/saath-aath/
cd saath-aath
npm install
gulp serve-dev
````
